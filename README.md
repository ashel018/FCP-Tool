# Fixed Program Chart (FPC) Automation Tool 

## Overview

The Fixed Program Chart (FPC) Tool is designed to automate program scheduling by integrating weekly FPC sheets with monthly Program Calendar data.

The application reduces manual effort by combining both datasets into a single structured workflow for easier tracking, validation, and scheduling management.

## Problem Statement

Program scheduling often requires managing multiple Excel sheets:

1. Fixed Program Chart (FPC)
   - Contains weekly scheduling details.

2. Program Calendar
   - Contains complete monthly program data.

Manual comparison and updates between these sheets can be time-consuming and error-prone.

## Solution

This tool combines both sheets automatically and processes scheduling information efficiently.

## Features

- Combines Weekly FPC Sheet with Monthly Program Calendar.
- Automated Scheduling Workflow.
- Data Validation and Processing.
- Reduces Manual Errors.
- Faster Program Planning.

## Tech Stack

- Python
- Excel Processing (Pandas / OpenPyXL)
- Flask (if UI available)
- HTML / CSS (if applicable)

## How It Works

1. Upload or load the FPC weekly sheet.
2. Load the Program Calendar monthly sheet.
3. Tool processes and merges scheduling data.
4. Outputs organized program information.
